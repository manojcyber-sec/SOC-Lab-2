# SOC Lab 2 – Log Analysis

## Overview
This project demonstrates basic system log analysis using journalctl in Kali Linux.

## Tools Used
- Kali Linux
- journalctl

## Steps Performed
1. Retrieved system logs using journalctl
2. Observed system activities and processes
3. Analyzed timestamps and system messages

## Key Observations
- System logs contain timestamps and process information
- Observed repeated system messages from background processes
- Identified normal system activity

## Learning Outcome
This project helped me understand how system logs are used in monitoring and analysis within a SOC environment.
